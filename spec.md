# Spec: [MANUAL] 계산기 나머지 연산 추가

## 배경 / 목적
현재 계산기 API는 +, -, *, / 연산을 지원한다.
나머지 연산(%)을 공식 지원하여 API 명세에 포함시키고,
정상 동작 및 엣지 케이스를 테스트로 보장한다.

## 범위 (In Scope)
- `CalculatorService`의 `%` 연산자 처리에 대한 단위 테스트 작성
- 0으로 나머지 연산 시 예외 처리 테스트
- `POST /api/calculate` 엔드포인트에서 `%` 연산자 통합 테스트
- 히스토리에 나머지 연산 결과가 저장되는지 확인 테스트

## 비범위 (Out of Scope)
- 서비스 로직 자체 변경 (이미 구현됨)
- 정수 나머지 연산 별도 지원 (Double 타입 유지)

## 인수 조건 (Acceptance Criteria)
- [ ] AC1: `10 % 3` 계산 시 결과값 `1.0`을 반환한다
- [ ] AC2: `10.5 % 3.0` 계산 시 올바른 부동소수점 나머지를 반환한다
- [ ] AC3: `b = 0`으로 나머지 연산 시 `ArithmeticException`을 던진다
- [ ] AC4: 나머지 연산 성공 시 히스토리에 저장된다
- [ ] AC5: `POST /api/calculate` API에서 `%` 연산자가 정상 동작한다

## 비기능 요구사항
- 기존 +, -, *, / 테스트에 영향 없어야 함
